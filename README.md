Ini adalah Project pertama saya yaitu Sistem Dashboard Karunia Kos

Project ini dibuat dengan kombinasi antara TypeScript, Tailwind css, dan vite.

## Fitur Utama

- **Dashboard Ringkasan:** Menampilkan total kamar, jumlah kamar terisi vs kosong, serta total pendapatan bulan ini.
- **Manajemen Kamar:** Mengedit status kamar (Tersedia, Terisi, Perbaikan), harga sewa, dan fasilitas kamar.
- **Pendataan Penghuni:** Mencatat identitas penghuni kos (Nama, No. HP, Tanggal Masuk) dengan aman.
- **Sistem Catatan Pembayaran:** Memantau riwayat pembayaran sewa bulanan dan mendeteksi anak kos yang belum bayar (tenggat waktu).
- **Integrasi WhatsApp:** Fitur tombol cepat untuk menghubungi penghuni langsung ke WhatsApp pemilik kos jika ada tagihan.
